Our code is on GitHub

GitHub is our favorite code hosting website, it has lots of great tools like 'issues' and 'pull requests' which help us develop stuff easier. SuicSoft uses GitHub to host their code (and Suici is friends with the Octocat!)

A Nice Little License

The SuicSoft Open License (SOL) is a short license which makes your code open to everyone, but doesn't allow selling your code, making stuff from your code not open source, creating malware from your code and copying the design.

About SuicSoft

SuicSoft makes free and open-source software for Windows PCs. Our free software use a the open-source license, SOL, and do not have malware, adware, spyware and all those kinds of badware
